### 介绍

cewl是一个专门为信息收集设计的命令行工具，用于从目标网站生成定制化的单词列表，这些单词列表通常用于密码工具（如字典攻击）或进行更多的社会工程学攻击。



### 基本功能

cewl会抓取目标网站上的内容（HTML页面），提取其中的关键词，并根据指定规则生成一个单词列表）





### 基本用法



#### 从目标url生成单词列表

```
cewl <url>
```

生成的单词列表会直接输出到终端





#### 将单词列表保存到文件

```
cewl <url> -w <文件名>
```





#### 自定义UA头

```
cewl <url> --ua "<UA>"
```
